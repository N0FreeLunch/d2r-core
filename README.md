## Verification Tools

You can use the following tools to verify and inspect save files.

- `d2save_verify`: Verifies the integrity of a `.d2s` save file.
- `d2save_map`: Displays a map of a `.d2s` save file.
- `d2save_diff`: Diffs two `.d2s` save files.
- `d2item_inspect`: Inspects the contents of a `.d2i` item file.
- `d2item_blob_inspect`: Inspects a blob of item data from a save file.
- `d2save_inject`: Injects data into a `.d2s` save file.
- `d2item_extract`: Extracts an item from a `.d2s` save file.
- `d2save_item_diff`: Diffs the items in two `.d2s` save files.
- `d2save_grid`: Displays the inventory grid from a `.d2s` save file.
- `d2save_hex_tail`: Shows the hex tail of a `.d2s` save file.
- `d2save_inventory_check`: Performs an inventory check on a `.d2s` save file.
- `d2save_bit_analyze`: Analyzes the bitstream of a `.d2s` save file.
- `dump_flags`: Dumps flag information from a file.
